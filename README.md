# Terraform Azure & AWS Foundation

This repository contains Terraform code for provisioning foundational
infrastructure on **Microsoft Azure** and **Amazon Web Services (AWS)**.

## Scope
- Azure: Resource Groups, VNet, Subnets, NSGs
- AWS: VPC, Subnets, Security Groups
- Modular Terraform design
- Secure and reusable patterns

## Tools
- Terraform
- Azure Provider
- AWS Provider
